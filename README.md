# 🧠 Wikipedia Text Preprocessing & Visualization

> Cleaning, transforming, and visualizing raw Wikipedia text data using NLP techniques.

## 🚀 Project Overview

This project focuses on preprocessing and analyzing textual data obtained from Wikipedia. The goal is to transform raw text into a clean and meaningful format and extract insights through visualization techniques.

## ⚙️ Project Steps
    1️⃣ Text Preprocessing
    2️⃣ Text Analysis
    3️⃣ Visualization
    4️⃣ Modular Function

## 🎯 Objectives

- Clean and normalize raw text data  
- Remove unnecessary components (punctuation, numbers, stopwords)  
- Identify and eliminate rare words  
- Apply tokenization and lemmatization  
- Visualize word frequencies and distributions  

## 📊 Dataset

The dataset contains textual data extracted from Wikipedia articles.

   **text**: Raw Wikipedia text content  

## 🛠️ Technologies Used

- Python 
- Pandas  
- NLTK  
- TextBlob  
- Matplotlib  
- WordCloud  

## 💡 Key Insights

- Text preprocessing significantly improves data quality  
- Removing rare words reduces noise  
- Visualization helps identify dominant terms quickly
